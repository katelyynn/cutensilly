bwaa (beta)

last.fm

bwaa brings last.fm back to 2012 while retaining all modern features, while including extra fun things:

  • in-built style loading with no need for other extensions
  • ‘recent activity’, a local! record of your last 10 site activities
  • support for more 2010-styled shouts, with the hope of more alternatives like it in the future
  • in-built dark theme ‘paint it black’
  • future-in-mind translation system (only english officially supported for now, i only speak english, my bad)

what’s planned?

  • bringing over bleh’s artist/album/track capitalisation correction system
  • more era-based alternatives within the settings
  • finishing some lesser-used pages, such as home, tags, events, etc.
  • also finishing more used pages (but harder to tackle!) such as charts (listening reports)
  • expanding the ‘recent activity’ system to track more
  • whatever else is tracked on the github page
